常用的Windows管理命令

记载常见的Windows管理命令，方便日常使用。

Windows操作系统提供了许多命令行工具，可以帮助用户进行各种系统管理和维护工作。除了netplwiz命令外，还有很多有用的命令。下面列举了一些常用的Windows命令及其用途：

系统信息和状态

  1. ipconfig

    • 用于显示所有网络接口的IP配置信息。
    • 示例：ipconfig /all 显示详细的网络配置。

  2. ping

    • 用于测试网络连接是否可达。
    • 示例：ping www.example.com 测试与www.example.com的连通性。

  3. nslookup

    • 用于查询DNS记录。
    • 示例：nslookup www.example.com 查询域名解析结果。

  4. tracert

    • 用于跟踪数据包到达目标主机所经过的路径。
    • 示例：tracert www.example.com 显示到达目标的路由。

  5. netstat

    • 显示活动的TCP/IP连接和侦听端口。
    • 示例：netstat -ano 显示所有活动连接及PID。

  6. tasklist

    • 列出当前运行的所有进程。
    • 示例：tasklist 显示所有正在运行的进程列表。

  7. taskkill

    • 终止指定的进程。
    • 示例：taskkill /im notepad.exe 终止记事本进程。

  8. msinfo32

    • 显示系统配置信息。
    • 示例：msinfo32 显示详细的系统信息。

文件和目录管理

  1. dir

    • 列出当前目录下的文件和子目录。
    • 示例：dir 列出当前目录的内容。

  2. cd (chdir)

    • 更改当前目录。
    • 示例：cd C:\Windows 进入Windows目录。

  3. mkdir

    • 创建新目录。
    • 示例：mkdir newfolder 创建名为newfolder的新目录。

  4. rmdir

    • 删除空目录。
    • 示例：rmdir newfolder 删除名为newfolder的目录。

  5. del (delete)

    • 删除文件。
    • 示例：del file.txt 删除file.txt文件。

  6. move (ren)

    • 移动或重命名文件。
    • 示例：move oldname.txt newname.txt 重命名文件。

  7. copy

    • 复制文件。
    • 示例：copy file1.txt C:\Backup 复制文件到指定目录。

  8. xcopy

    • 复制文件和目录树。
    • 示例：xcopy C:\Source\*.* C:\Destination 复制整个目录树。

用户账户管理

  1. net user

    • 管理本地用户账号。
    • 示例：net user username password /add 添加新用户。

  2. net localgroup

    • 管理本地组成员。
    • 示例：net localgroup administrators username /add 将用户添加到管理员组。

  3. whoami

    • 显示当前用户的用户名和用户组信息。
    • 示例：whoami 显示当前用户的详细信息。

  4. gpedit.msc

    • 打开本地组策略编辑器。
    • 示例：gpedit.msc 调用组策略编辑器。

  5. secpol.msc

    • 打开本地安全策略。
    • 示例：secpol.msc 调用安全策略编辑器。

服务管理

  1. services.msc

    • 打开服务管理控制台。
    • 示例：services.msc 调用服务管理界面。

  2. net start

    • 查看启动的服务。
    • 示例：net start 显示所有启动的服务。

  3. net stop

    • 停止服务。
    • 示例：net stop service_name 停止指定服务。

  4. sc (service control)

    • 控制服务。
    • 示例：sc query service_name 查询服务状态。

其他实用命令

  1. cmd

    • 打开命令提示符窗口。
    • 示例：cmd 打开一个新的命令提示符窗口。

  2. regedit

    • 打开注册表编辑器。
    • 示例：regedit 调用注册表编辑器。

  3. powershell

    • 打开PowerShell窗口。
    • 示例：powershell 打开PowerShell命令行。

  4. eventvwr.msc

    • 打开事件查看器。
    • 示例：eventvwr.msc 调用事件查看器。

  5. diskmgmt.msc

    • 打开磁盘管理工具。
    • 示例：diskmgmt.msc 调用磁盘管理界面。

  6. diskpart

    • 管理磁盘分区。
    • 示例：diskpart 进入磁盘分区管理命令行。

  7. sfc /scannow

    • 扫描并修复系统文件。
    • 示例：sfc /scannow 检测并修复损坏的系统文件。

这些命令可以帮助您更好地管理和维护Windows系统。当然，这只是其中的一部分，还有许多其他命令可以探索和使用。如果您需要特定的功能或操作的帮助，请告诉我，我可以为您提供更详细的指导。
